Cwicz.4

Inne tagi w HTML

Ty zaproponuj swoje, związane z informatyką

mozliwisc formowania w HTML:

  • emfaza czcionki: < em > < / em >
  • zmniejszenie czcionki: < small > < / small >
  • tabela: < table > < / table >

Jeszcze:

  1. nagłówek sekcji zawierającej wiele śródtytułów < hgroup > < / hgroup >
  2. etykieta kontrolki formularza < label > < / label >
  3. stopka strony lub fragmentu dokumentu < footer > < / footer >
  4. reprezentacja postępu zadania < progress > < / progress >
